# Apa itu Java ?

Java adalah sebuah bahasa pemrograman yang populer dan powerful, ia kembangkan
pada tahun 1955 oleh James Gosling. penamaan JAVA sendiri terinspirasi dari
jenis kopi yang berasal dari Indonesia.

Java memiliki jargon "Tulis satu kali jalankan dimana saja" yang artinya bisa
Run dari perangkat apapun. kini di perkirakan ada lebih dari 3 milyar perangkat
yg berjalan di atas Java.

# Java dapat digunakan untuk :

- Aplikasi mobile terutama Android
- Aplikasi desktop ( PC )
- Aplikasi web
- Game
- Database
- Dan lain-lain

# Mengapa kamu memilih menggunakan Java ?

- Java berjalan di multi platform ( Windows, Mac, Linux, Raspberry Pi, Dan
  lainya )
- Java sangat populer, bahkan sempat digunakan juga pada event NASA Mars Rover
  untuk keperluan visualisasi data dan beberapa software dalam perangkat robot
  tersebut
- Java programmer memiliki jumlah permintaan yang sangat besar dalam job market
- Open source
- Memiliki banyak support dan komunitas, termasuk di Indonesia
- Java adalah bahasa pemrograman yang berorientasi pada objek atau OOP: Object
  Oriented Programming
- Java memiliki gaya kode yang tidak asing dari kelas bahasa C, C++, C#
